# dotfiles
My i3-Gaps Configuration 


## Setup
- Distribution: `Arch Linux`
- Application Launcher: `rofi`
- Shell: `zsh`
- Terminal Emulator: `urxvt`
- Text Editor: `vim`
- Window Manager: `i3-Gaps`
